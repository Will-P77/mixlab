# Firebase browser modules

These files are the official Firebase Web SDK 12.16.0 ESM browser modules, downloaded from Google's documented `gstatic.com/firebasejs` distribution. The Auth and Firestore module imports were changed only to reference the local `firebase-app.js` file.
